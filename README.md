# FreshMall-pc

# 鲜果优品pc端 [在线预览](http://www.justic.store:8083/#/index)

# 一、项目介绍

基于vue.js的生鲜线上零售平台。

## 二、技术栈

- vue
- vue-cli

- vue-router
- vuex

- axios
- webpack

- ES6
- element-ui

- sass

## 三、如何运行该项目

```javascript
// 进入该项目
cd FreshMall-pc

// 安装对应依赖
npm install

// 访问网站
npm run serve
```
